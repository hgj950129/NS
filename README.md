# NS
使用重力传感器控制小球的移动,在SurfaceView中添加一个小球的图片；调用重力传感器，获取重力传感器的X，Y轴的值，并将其传到小球的X,Y坐标，然后以50帧的速率不断重画SufaceView,实现重力传感器控制小球的功能。
